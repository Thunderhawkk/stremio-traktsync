Frequently asked questions

Do I need a Trakt account?

Yes. Sign in to Trakt once from the dashboard, and the add‑on will import lists using your authorization.

Where do my catalogs show in Stremio?

Inside Stremio’s Catalog section as a custom group label (from your Catalog Prefix) with each list as a selectable catalog.

Is my manifest private?

Yes. Each user gets a tokenized manifest URL; revoke or rotate the token from the dashboard at any time.

How are tokens handled?

Access tokens refresh automatically in the background before expiry, keeping sessions active without prompts or manual re‑auth.

Can I filter and sort large lists?

Yes. Catalogs are server‑paginated with filters for year, rating, runtime, and genre, plus sorting by rating, year, runtime, or name.

How do I install the add‑on?

Copy the personal manifest URL from the dashboard and paste it in Stremio: Add‑ons → Install via URL, then select your custom group.
